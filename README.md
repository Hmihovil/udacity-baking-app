# Udacity Baking App
This project is the third task of the Android Nanodegree Scholarship.

<b>Please note;</b> Base URL is hidden due to the security purpose. It should be placed in to `gradle.properties.`

# What the project uses

* MVP Pattern
* BindingAdapter
* Retrofit2
* RxJava2
* Glidev4
* GlideModule
* Timber
* Okhttp
* DataBinding
* SQLite
* Schematic
* Widgets
* ExoPlayer
